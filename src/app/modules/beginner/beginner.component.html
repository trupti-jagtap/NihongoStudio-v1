Syllabus for Class 1 : Basic First Half N5 level

  • Basic class contents

    This class is recommended for beginners with no or little previous knowledge of Japanese. Study the basic Japanese grammar and words with the main focus on the oral communication skill. The goal of this course is to enable you to conduct daily conversations.


  • Objectives

    • Acquire basic conversational skills for greeting, self-introduction

Syllabus for Class 2 : Basic Second Half N4 level

  • Basic class contents

    Study the basic Japanese grammar and words with the main focus on the oral communication skill. The goal of this course is to enable you to conduct the daily conversations.


  • Objectives

    • Learn to converse in a simple but clear manner in most situations that the student may encounter at schools, shops, or banks.
    • Learn to talk with friends in Japanese.
    • Learn to read and write simple sentences in Japanese using a few Kanji with additional supporting materials.
